# AWS Live Church Translation Project

## Overview
This project provides live Japanese-to-English subtitle translation for church services using AWS.

## Features
- Live Japanese speech recognition
- Japanese subtitle display
- English translation display
- AWS Lambda backend
- Amazon Translate integration
- Projector-ready browser display

## Technologies Used
- HTML
- CSS
- JavaScript
- AWS Lambda
- Amazon API Gateway
- Amazon Translate

## Architecture
Browser Microphone → API Gateway → Lambda → Amazon Translate → Browser Display

## Future Improvements
- Voice-to-voice translation
- Mobile application
- Multi-language support
- Speaker diarization
- Subtitle saving
